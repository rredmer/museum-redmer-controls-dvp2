## Introduction

The DVP2 Digital Printer Controller is a software solution developed by Redmer Controls for controlling ISA digital printers. This project includes various components and configurations necessary for deploying and running the DVP2 system.

<img src="Docs/2004 Redmer Controls - DVP2.jpg" width="400" />

## Project Structure

The project is organized into the following directories:

- **Calibration Images/**: Contains calibration images used for printer setup and testing.
- **Database/**: Contains database files for storing settings and configurations.
- **Docs/**: Documentation related to the project.
- **Drawings/**: Contains drawings related to the project.
- **DVP2_Printers/**: Contains printer-specific configurations and resources.
- **QC Stations/**: Contains quality control station configurations and resources.
- **Setup/**: Contains setup files for installing the DVP2 system.
- **Src/**: Source code for the DVP2 system, organized into various modules:
  - **Picto/**: Contains pictographic resources.
  - **Pictographics/**: Contains pictographic resources.
  - **Printer/**: Contains printer-related source code.
  - **QC/**: Contains quality control-related source code.
  - **Shared/**: Contains shared resources and utilities.

## Installation

To install the DVP2 system, follow these steps:

1. Navigate to the `Setup/` directory.
2. Run the `setup.exe` file to start the installation process.
3. Follow the on-screen instructions to complete the installation.

## Usage

After installation, you can start the DVP2 system by running the main executable file. The main application interface is provided by the `MainForm` module in the `Src/Printer/` directory.

## Configuration

Configuration files for the DVP2 system are located in the `Src/` directory. Key configuration files include:

- **DVP2_QC.PDM**: Configuration for the quality control stations.
- **dvp2.PDM**: Configuration for the printer deployment package.

These files contain various settings and dependencies required for the DVP2 system to function correctly.
